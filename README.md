# Currency Converter (Конвертер валют)

GUI-приложение для конвертации валют с использованием внешнего API.

## Автор
**Фаяз Гилязов** (замените на ваше имя)

## Описание
Приложение позволяет конвертировать суммы между различными валютами, используя актуальные курсы от exchangerate-api.com. История конвертаций сохраняется в JSON-файл.

## Требования
- Python 3.6+
- Библиотеки: requests, tkinter (встроен в Python)
